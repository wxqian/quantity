# 策略规范指南 (Strategy Specs)

本目录用于定义具体交易策略的配置参数、逻辑流程和绩效目标。

## 规范内容

一个完整的策略规范应包含：
1.  **策略元数据**: ID, 名称, 作者。
2.  **参数定义**: 策略运行所需的外部参数（名称、类型、默认值、范围）。
3.  **交易逻辑**: 
    *   **入场信号**: 什么条件下买入？
    *   **出场信号**: 什么条件下卖出？
    *   **风控规则**: 止损、止盈、最大持仓等。
4.  **回测要求**: 使用什么数据集、时间段进行验证。

## 模板 (Template)

### [Strategy-001] 双均线趋势策略

#### 1. 参数定义 (Parameters)

| 参数名 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| `symbol` | string | "000001.SZ" | 交易标的 |
| `fast_window` | int | 5 | 快线周期 |
| `slow_window` | int | 20 | 慢线周期 |
| `volume` | int | 100 | 单笔下单数量 |

#### 2. 逻辑 (Logic)
*   **计算指标**:
    *   `MA_Fast` = 收盘价的 `fast_window` 周期简单移动平均。
    *   `MA_Slow` = 收盘价的 `slow_window` 周期简单移动平均。
*   **入场 (Buy)**:
    *   当 `MA_Fast` 上穿 `MA_Slow` (金叉) 且 当前无持仓 -> **买入** `volume` 股。
*   **出场 (Sell)**:
    *   当 `MA_Fast` 下穿 `MA_Slow` (死叉) 且 当前持仓 > 0 -> **卖出** 全部持仓。

#### 3. 风险控制 (Risk Control)
*   **每日最大回撤**: 2%。
*   **单笔止损**: 无 (仅跟随均线交易)。
